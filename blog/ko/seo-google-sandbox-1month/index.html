결론 먼저: Google sandbox 는 실재한다. 신규 도메인 → 색인은 점진적으로 됨 (5/22 부터 spike) 그러나 검색 노출은 spike → dip 패턴 (5/14 정점 → 5/24 거의 0). Naver·Cloudflare 가 받쳐주지 않았으면 패닉했을 상황. 6-12주 자동 종료 expected.

배경

taystudio.github.io → taystudios.com (2026-05-09 migration). 신규 도메인 취급. 5/9 이후 30일간 GSC·Naver·Cloudflare 3종 데이터 추적 + 무슨 일이 일어났는지.

Google Sandbox 가 뭔지

Google 이 신규 도메인을 일정 기간 "견습" 상태로 두는 비공식 알고리즘: - 6-12주 색인은 되지만 검색 순위 상승 제한 - 스팸·MFA(Made For AdSense) 사이트 차단 목적 - "이 사이트가 6주 후에도 살아있나" 평가 - Google 공식 부정하지만 SEO 업계에서 widely observed

Bing Webmaster

비슷한 패턴: - 5/12 까지 데이터 있음 (옛 도메인 잔여) - 5/22 부터 sitemap discovered 124 URLs - 그러나 검색 노출 0 (Bing 도 신규 도메인 평가 큐 대기)

Naver SearchAdvisor (5/22 등록)

여기서 반전: - 5/22 등록 직후 spike 시작 - 5/27 정점 7,600 노출 - 5/28 이후 일 평균 2,000-3,000 안정 - 30일 합계: 노출 26,000 + 클릭 230

= Google 의 33-115배. 한국 사용자 대상이라 Naver 매칭이 훨씬 좋음.

Cloudflare Web Analytics

전체 트래픽 (모든 채널): - 일 visits 평균 55명 - 79% 가 Naver/Daum referrer - ChatGPT referrer 일부 (AI 검색 노출 시작) - 사이트 자체는 정상 작동

발견 1 — Google sandbox 는 "검색 순위" 만 영향

색인 자체는 점진적으로 OK: - 5/22 122 페이지 색인됨 - 5/29 색인 안 된 페이지 7건 (redirect 4·canonical 1·큐 2 = 정상 사유) - 6/2 sitemap 124 URL 모두 발견

= Google 이 사이트 발견·크롤·색인은 정상 진행. 검색 결과 페이지의 순위만 제한. 같은 키워드에 다른 사이트 (오래된 도메인) 가 우선.

발견 2 — 5/14 spike 의 정체

5/14 갑자기 노출 450 → Google 이 신규 도메인에 테스트 노출 한 듯. 사이트가 진짜인지 사용자 반응 (CTR·체류시간) 확인.

그 후 5/17~24 dip = "평가 결과 일단 sandbox 유지". 클릭률 0.3% 가 신규 도메인의 평가 기준 못 미친 것으로 추정.

발견 3 — Naver 는 sandbox 같은 평가 없음

5/22 등록 → 즉시 색인 + 노출 시작. 3일만에 일 노출 2,000+ 도달. Naver 는 sitemap 제출되면 빠르게 처리. 신규 도메인이라도 큰 페널티 X.

발견 4 — AI 검색이 새로운 채널

Cloudflare 에서 chatgpt.com referrer 출현. LLM (ChatGPT·Claude·Perplexity) 이 우리 사이트 학습·검색에서 인용 시작. Google sandbox 와 무관한 채널.

추정 — Google sandbox 종료 시점

도메인 age sandbox 상태 검색 노출
1주 (5/16) 초기 평가 spike 시작
2주 (5/22) 평가 진행 spike 정점
3주 (5/29) dip 거의 0
6주 (6/20) 부분 회복 시작 expected 노출 점진 증가
12주 (7/30) 전체 회복 expected 정상 노출

이 timeline 은 일반 SEO 경험치 (Google 공식 X). 콘텐츠 quality·E-E-A-T·백링크에 따라 단축 가능.

우리 액션 — sandbox 시기에 한 일

콘텐츠 강화

  • 18 도구 stale fix (잘못된 수치 24건)
  • 차별화 매트릭스 5건 add (inheritance break-even·loan DSR·6+6 부모공동 등)
  • Top 5 페이지 title·description CTR 최적화
  • 블로그 글 시리즈 시작

검색엔진 분산

  • Naver SearchAdvisor 등록 + sitemap 제출
  • Bing Webmaster URL submission (10건)
  • Daum Webmaster 예정
  • llms.txt + robots.txt LLM crawler Allow (AI 검색 분산)

결과 (1개월 차)

  • Google: sandbox 진행 중 (대기)
  • Naver: 일 visits 50+ 안정
  • Bing: sitemap discovered, 크롤 대기
  • AI 검색: 일부 referrer 출현

운영자 교훈

1. 도메인 migration 전 정확한 expectation

신규 도메인 = 검색 0-3개월 노출 안 됨. 비즈니스가 검색 트래픽 의존이면 migration 신중하게.

2. GSC 만 보면 망함

데이터 한쪽만 보고 패닉 X. Naver + Cloudflare + Bing 합쳐서 봐야.

3. 콘텐츠 차별화는 sandbox 와 무관

매트릭스·시뮬레이션·시각화 같은 차별화 콘텐츠는 long-tail SEO 에 누적. sandbox 종료 후 즉시 노출 ↑.

4. AI 검색 노출 시작은 별개 채널

ChatGPT·Claude·Perplexity 가 인용 시작 → 미래 트래픽. llms.txt 적극 활용.

한 줄 결론

Google sandbox 는 진짜다. 그러나 한국 사이트면 Naver 가 받쳐준다. 패닉 X, 시간 + 콘텐츠 + 채널 분산.

7월 말 ~ 8월에 다시 회고 글 작성 예정. sandbox 정말 종료되는지 실측 데이터로.
