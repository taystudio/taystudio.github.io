개인정보 처리방침

최종 업데이트: 2026-05-31

한 줄 요약

TAYSTUDIO Blog 는 방문자 식별 정보를 자체적으로 수집하지 않습니다. 댓글 기능(giscus)을 사용할 때만 GitHub 계정 정보가 GitHub 시스템에 처리되며, 그 외 모든 동작은 사용자 브라우저 안에서 끝납니다.

1. 수집하지 않는 것

  • 이름·이메일·전화번호·주소·생년월일 등 개인식별 정보
  • IP 주소·접속 로그·페이지 체류 시간 (서버 분석 도구 미사용)
  • 쿠키 (자체 설정 X — 브라우저 localStorage 만 일부 기능에 사용)

2. 사용자 브라우저에서만 처리되는 것 (localStorage)

다음 기능을 위해 사용자 본인 브라우저에만 저장됩니다. 운영자는 접근 불가:

키 용도
taystudio.theme 다크·라이트 모드 선택
tayblog_cat_collapsed / _en 사이드바 카테고리 접힘 상태

브라우저 설정 → 사이트 데이터 삭제로 언제든 초기화 가능합니다.

3. 댓글 기능 — giscus

댓글은 giscus.app 을 통해 GitHub Discussions 에 저장됩니다.

  • 댓글 작성 시 GitHub 로그인 필요 — GitHub OAuth 인증으로 본인 계정과 연결.
  • 저장 위치: 공개 GitHub Discussions (운영자 리포지토리). 누구나 볼 수 있는 공개 데이터.
  • 삭제 요청: GitHub 계정으로 본인 댓글 직접 삭제 가능. 어려운 경우 아래 연락처로.
