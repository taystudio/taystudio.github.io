🐛 Descender clearance fix (사용자 신고)

"Useful things" 타이틀에서 "g" 아래 부분 잘림. 원인 = line-height: .96 + -webkit-background-clip: text 조합. line-height < 1이면 element box가 descender보다 짧고, 그 영역 밖은 background gradient 안 그려져서 transparent 처리.

.hub-title {
  ...
  line-height: .96;
  /* descender(g·j·p·q·y) clearance */
  padding-bottom: 0.18em;
  background: linear-gradient(...);
  -webkit-background-clip: text; background-clip: text;
  -webkit-text-fill-color: transparent;
