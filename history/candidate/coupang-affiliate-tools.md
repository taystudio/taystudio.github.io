## 🥇 톱 2 — 가장 빠른 시작

### 1️⃣ 저장공간 계산기 + SSD/외장하드 추천 (`storage/` 또는 `tools/storage/`)

**왜 best**:
- 작업 매우 작음 (1~2시간) — 계산식 + 쿠팡 카드 4~6개
- SERP 빅키워드 = "4K 영상 용량"·"외장하드 추천"·"SSD 1TB 가격"·"SD카드 용량 계산"
- 단가 큼 (외장하드 5~30만, SSD 10~50만)
- 우리 video/ 도구 사용자와 자연 연결 (영상 압축 후 저장 → 외장하드)

**도구 구조**:
```
입력 매트릭스 (탭 또는 select):
  A. 영상: 시간 (1h) + 해상도 (4K/QHD/FHD) + 코덱 (H.264·H.265)
  B. 사진: 수 (5000장) + 종류 (RAW·JPG·HEIC) + 평균 사이즈
  C. 일반: 직접 GB 입력
       ↓
계산:
  영상: bitrate × 시간 (4K H.264 ≈ 50Mbps × 3600s ÷ 8 ≈ 22GB/h)
  사진: 수 × 평균 (RAW 30MB·JPG 5MB)
       ↓
출력: "필요 용량 ≈ 120GB"
       ↓
추천 카드 (쿠팡 4~6개):
  - 1TB 휴대용 SSD (삼성 T7 ~10만)
  - 2TB 외장 HDD (Seagate ~8만)
  - 1TB NVMe 내장 (WD Black ~15만)
  - 256GB SD카드 (SanDisk Extreme Pro ~5만)
  - 4TB 외장 HDD (대용량 백업)
  - 클라우드 비교 (Google One·iCloud — 정보용·쿠팡 없음)
```

**SEO 키워드**:
- title: "저장공간 계산기 | 영상·사진 용량·외장하드 SSD 추천"
- keywords: 저장공간 계산기, 4K 영상 용량, 외장하드 추천, SSD 1TB 가격, SD카드 용량, 영상 GB 계산, 백업 용량 계산
- 차별화 = 영상 비트레이트·사진 종류별 정확한 추정 + 본 사이트 video/ 도구와 연계

**inline 표 (콘텐츠 깊이)**:
| 영상 종류 | 1시간당 용량 |
|---|---|
| 4K H.264 (50Mbps) | ~22GB |
| 4K H.265 (25Mbps) | ~11GB |
| FHD H.264 (8Mbps) | ~3.6GB |
| 폰 4K (60Mbps 평균) | ~27GB |

| 사진 종류 | 평균 사이즈 |
|---|---|
| DSLR RAW | 30MB |
| 미러리스 RAW | 25MB |
| 폰 HEIC | 2~3MB |

### 2️⃣ 모니터 해상도·인치 변환 + 모니터 추천 (`tools/monitor/` 또는 `monitor/`)

**왜 best**:
- 모니터 단가 매우 큼 (30~150만)
- "모니터 인치 시야 거리"·"DPI 계산"·"4K vs FHD 27인치" SERP 큼
- 정적 0KB
- 본인이 모니터 살 직전 = 구매 의도 직접

**도구 구조**:
```
입력:
  - 인치 (24·27·32·34 ultrawide·43)
  - 해상도 (FHD 1920×1080·QHD 2560×1440·4K 3840×2160)
  - 시야 거리 (50·70cm — 권장 거리 자동 계산도)
       ↓
계산:
  - DPI = √(가로² + 세로²) ÷ 인치
  - 픽셀 사이즈 (mm)
  - 권장 시야 거리 (Apple HIG·VESA 표준)
  - 각도(°)별 시야각
       ↓
판정: "QHD 27인치는 50cm 거리에서 적정 (DPI 109, 픽셀 0.23mm)"
       ↓
사용 시나리오 추천:
  - 게이밍 (144Hz+ · 1ms · IPS Fast)
  - 작업·디자인 (4K IPS · sRGB 99%+ · USB-C 90W)
  - 듀얼 모니터 세트 (24인치 × 2)
  - 코딩 (세로 모드 · 울트라와이드 34")
       ↓
쿠팡 카드 (사용 시나리오별 4~6개):
  - 게이밍 27인치 QHD 165Hz (LG·삼성 ~50만)
  - 작업 27인치 4K IPS (LG·DELL ~70만)
  - 듀얼 24인치 FHD (~25만 × 2)
  - 울트라와이드 34인치 QHD (~70만)
  - 휴대용 15.6인치 (~25만)
```

**SEO 키워드**:
- title: "모니터 해상도·인치 가이드 | DPI 계산·시야 거리·추천"
- keywords: 모니터 인치, 모니터 추천, 4K 모니터, QHD 27인치, 게이밍 모니터, DPI 계산, 모니터 시야 거리, 듀얼 모니터, 울트라와이드

**inline 표** — DPI vs 인치/해상도 매트릭스 (논리적·SEO 효과 큼):
| 인치 \ 해상도 | FHD | QHD | 4K |
|---|---|---|---|
| 24" | 92 (적정) | 122 (선명) | 184 (텍스트 너무 작음·스케일링 필요) |
| 27" | 82 (텍스트 큼·약간 흐림) | 109 (sweet spot) | 163 (스케일링 125%) |
| 32" | 69 (작업용 X) | 92 (적정) | 138 (스케일링 100%) |

### 🥈 다음 후보 — baby/ 또는 pet/ 카테고리 진입

분유·사료는 **재구매 단가** 큰 영역. 한 번 사면 끝나는 모니터·외장하드와 달리 매월 결제 = LTV 좋음.

#### baby/ — 분유 양·기저귀 사이즈

```
도구 구조:
  - 분유 양 계산: 출생일 + 체중 → 일일 분유 ml + 횟수 (KCDC·소아과 표준)
  - 기저귀 사이즈: 체중 → 사이즈 (S·M·L·XL — 한국 브랜드별)
  - 이유식 시기: 출생일 → 4개월·6개월·9개월 단계별 재료
  - 예방접종 일정표: 생년월일 → 자동 일정

쿠팡: 매일·아이배냇·하기스·팸퍼스·이유식 키트
SERP: "신생아 분유 양"·"기저귀 사이즈"·"이유식 시작 시기"
```

#### pet/ — 반려동물 사료 양

```
도구 구조:
  - 사료 양: 견종·체중·연령 → 일일 g (브랜드 가이드 평균)
  - 강아지·고양이 나이 변환: 사람 나이 환산
  - 식품 위험도: 양파·초콜릿·포도 등 (정보 도구)
  - 의료비 가이드: 중성화·스케일링 평균 비용

쿠팡: 로얄캐닌·내추럴발란스·오리젠·사료·간식·약
SERP: "강아지 사료 양"·"고양이 사료 g"·"강아지 나이"
```

## 진입 순서 권장

```
1. 쿠팡 파트너스 신청 (지금)
   → 결과 대기 1주~
2. 저장공간 계산기 1선 출시 (1~2h, 신청 결과 무관)
3. 모니터 가이드 1선 출시 (2~3h)
   → 쿠팡 승인 시 카드 삽입
4. CTR·전환 데이터 1~2주 관찰
5. 데이터 기반으로 baby/ 또는 pet/ 진입
```

## 트레이드오프·주의

### 정책
- 쿠팡 파트너스 = 광고 명시 의무: "이 게시물은 쿠팡 파트너스 활동의 일환으로, 일정액의 수수료를 제공받습니다." 페이지 하단 명시
- AdSense + 쿠팡 동시 가능. 단 광고 영역 시각 충돌 X 디자인
- 쿠팡 dynamic banner 사용 시 자동 갱신 (수수료 X 자체 디자인 OK)

### SEO·UX
- 추천 카드는 도구 결과 직후 자연스러운 흐름에 배치 — 결과 위·도구 영역 침범 X
- "이 정도 용량이면 1TB SSD 권장 → 쿠팡 link" 흐름이 best
- 카드 4~6개가 적정 — 더 많으면 광고 폭격 인상

### 신청 거부 시
- 트래픽 데이터 좀 쌓고 재신청 (1~2개월)
- 또는 AdSense 승인 후 재신청 (사이트 신뢰도 ↑ 시그널)
- 거부 사유는 메일로 옴 → 그에 맞춰 보강
