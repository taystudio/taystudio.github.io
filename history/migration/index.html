↻ Rollback 시나리오

치명적 문제 시 단계별 복귀. 양쪽 도메인 모두 작동 중이라 즉시 복구 가능.

상황증상대응
cert 발급 실패 https 인식 X · browser warning Settings → Custom domain 비우고 다시 입력 → DNS 재검증 → 30분~24h 대기
https↔http loop ERR_TOO_MANY_REDIRECTS Cloudflare SSL/TLS 모드 = Full 확인. Flexible이면 즉시 변경
sed 후 canonical 깨짐 HTML 태그 밸런스 fail git reset --hard HEAD~1 (push 전) 또는 revert commit (push 후)
SC 노출수 80%+ 폭락 (1주 후) 301 redirect 미작동 또는 Change of Address 누락 옛 도메인 페이지에서 curl -I로 301 확인. 안 나오면 GitHub Pages custom domain 재설정
IndexNow ping 실패 HTTP 403 / 422 {key}.txt curl 200 확인 → indexnow-ping.sh의 host 변수 갱신 재확인
전면 rollback (최후) 모든 신호 망가짐 Settings에서 custom domain 비우기 → 옛 도메인만 작동. SEO 권위 일부 손실 영구
