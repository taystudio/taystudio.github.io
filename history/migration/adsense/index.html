결정 의사결정 메모

검토 중 script 활성화 = 정공법

"승인 후 활성화" 시나리오 = 거의 100% 거절 (광고 코드 미발견). AdSense는 검토 시점에 광고 코드가 있어야 정상 진행. 옛 도메인이 "준비 중" 스턱이었던 직접 원인 = script 주석. 신규 도메인은 검토 요청 직후 script 활성화로 같은 함정 회피.

메타 태그 검증 방식 채택

AdSense가 발급한 두 검증 옵션 중 메타 태그 채택. 이유 = ① 다른 채널(GSC·Naver·Bing)과 head 메타 일관성 ② head 1줄 추가가 가장 가벼움 ③ ads.txt는 이미 있어 추가 작업 0건. 광고 송출용 script는 별도 작업.
