## 생존 후보 (5개 필터 모두 통과)

### 1순위 — 통상임금/평균임금 계산기
- 산식: 근로기준법 본문 박힌 법정 고정 (매년 갱신 0)
- internal SEO: 퇴직금·시급·실수령액·실업급여 도구의 **input value** → 5겹 link 자연 형성
- 신뢰 리스크: 회사·노무사 재확인 전제, disclaimer + 산식 공개로 충분
- evergreen: 근로기준법 거의 안 바뀜

### 2순위 (1순위 안정화 후 2~4주 뒤) — 야근수당/휴일근로 계산기
- 산식: 1.5배·2배·야간0.5배 법정 고정

### 핵심 패턴 — "유지보수 함정"

컷된 후보의 공통점: **매년 요율/공제율/상한 변경**.
"광고 CPC 최상위"에 끌려 합리화한 selection bias.
솔로 개발자 + 80개 기존 도구 + "공식과 1원 차이로 신뢰 즉사" 조건에선 SEO 기회가 아니라 **유지보수 함정**.

법정 산식이 본문에 박힌 evergreen 도구만 안전 (통상임금·야근수당이 그 예).

### 핵심 패턴 — "인생 의사결정 카테고리"

노후(국민연금)·출산(통합 시뮬)·대출(DSR)은 사용자가 그대로 믿고 결정. 1만원 misalign = AdSense·쿠팡 한 달 수익 < 신뢰 손상 비용. source accuracy strict 룰을 끝까지 지킬 자신 없으면 진입 금지.

## 외부 에이전트가 발견한 추가 후보 3개

### A. 이미지 합치기 (가로/세로 이어붙이기) ★
- 월 40k~70k — **카톡 대화 캡처·쿠팡 후기·인스타 9분할** = 한국 특유 use case
- Canvas drawImage 단순, dwell 2~4분
- **추정 $400~600/월**

### B. MP3 자르기 / 벨소리 만들기 ★
- 월 50k~80k — **iPhone 벨소리 30초 제한 + 카톡 알림음** evergreen
- Web Audio API + lamejs로 정적 가능, dwell **3~6분 (최장)**
- **추정 $500~700/월**
- 신규 audio 카테고리지만 video 인접이라 정체성 OK

### C. 유튜브 썸네일 다운로더 + 그리드 비교
- 월 30k~50k
- 단순 다운로드는 thin → **4해상도 비교 + 한국어 가독성 체커 + 그리드 묶음**으로 dwell 확보
- 추정 $300~500/월
- ToS 회색지대 (썸네일은 공개 리소스라 OK)

## 단일 최강 1순위 — WebP/AVIF ↔ JPG/PNG 변환

근거 (트래픽 × 체류 × 승인안전성 곱이 최대):
1. **AdSense 승인 안전성 1위** — 변환 결과는 명백한 산출물, thin content 의심 0
2. **트래픽 1위** — webp 강제저장은 카톡·인스타·네이버블로그 매일 발생, ChatGPT 해결 불가 (파일 처리)
3. **유지보수 0** — Canvas + 브라우저 native decoder만, F1 통과 가장 확실
4. **기존 image 11개와 cross-link 폭발** — compress·resize·heic-to-jpg와 "이미지 변환 허브" 형성

CPC는 PDF가 더 높지만, **승인 안전성 + 트래픽 + 유지보수 0** 3축 모두 1등.
