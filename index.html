정말 무료인가요? 숨겨진 비용은요?

전부 무료입니다. 회원가입·결제·이메일 입력 모두 없습니다. 사이트 운영은 화면에 표기된 광고·affiliate 추천(쿠팡 파트너스 등)으로만 이뤄지며, 어느 도구도 사용에 제한이 없습니다.

업로드한 파일·입력값은 어디로 가나요?

외부 서버로 전송하지 않습니다. 이미지·PDF·동영상 처리, 계산기 입력값 모두 사용자 브라우저 안에서만 처리되고, 결과 다운로드 후에는 메모리에서 사라집니다. 서버에 어떤 입력값도 저장하지 않으며 로그도 남지 않습니다.

통계는 익명 페이지뷰만 수집합니다 (Google Analytics). 파일 내용·계산 입력값은 GA로도 전송되지 않습니다.

계산기 결과가 실제 명세서·고지서와 다른데요?

본 사이트의 계산기는 공식 요율 기반 추정치입니다. 회사 비과세 정책(식대 20만원·자가운전 등), 회사 자체 공제(노조비·복지기금), 연말정산 시 추가 공제, 건강보험 4월 정산 등에 따라 ±10~30만원 차이가 날 수 있습니다. 각 도구 페이지 하단의 "공식 출처" 섹션에서 적용한 법령·요율의 출처 링크를 확인할 수 있습니다.

아이폰 HEIC 파일을 어디서 변환하나요?

이미지 → HEIC → JPG 도구에서 변환합니다. 다중 페이지 HEIC도 자동 분리되며, 20장 이상 batch 처리 시 confirm으로 안전망 표시. 변환된 JPG는 EXIF GPS도 제거된 상태로 저장됩니다.

iPad·iPhone에서도 잘 작동하나요?

iOS Safari 15+ / iPadOS 15+에서 정상 작동합니다. 카메라 권한이 필요한 도구(QR 인식·OCR)는 Safari가 권한을 요청하고, 권한 거부 시에는 갤러리에서 이미지를 업로드해 처리할 수 있습니다. iCloud·Google Photos 같은 클라우드 사진을 바로 불러올 때 권한 만료 안내가 나오면 다시 선택해 주세요.

큰 파일은 어디까지 가능한가요?

도구별로 가드가 다릅니다 — 이미지 100MB·PDF 100MB·동영상 1GB가 일반적인 상한입니다. 메모리·기기 성능에 따라 그보다 작아도 실패할 수 있어, 동영상 도구는 추정 메모리를 사전 표시하고 위험 시 confirm으로 차단합니다.

법령·요율이 바뀌면 어떻게 갱신되나요?

각 도구 페이지의 "공식 출처" 섹션에 최종 검증일을 표기합니다. 매년 11월(고시 시즌)·1월(연도 시작)에 전수 점검하며, 큰 변경(국민연금 인상, 최저시급 등)이 발표되면 즉시 반영합니다. 사이트 변경 이력은 About 페이지에서 확인할 수 있습니다.
