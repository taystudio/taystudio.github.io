운영자 소개

TAYSTUDIO는 실용 utility 도구 65+선을 무료로 제공하는 1인 운영 사이트입니다.

마지막 갱신: 2026-05-13

사이트 미션

직장인·자영업자·학부모·학생이 자주 필요로 하는 utility 작업을 브라우저 안에서 즉시 처리하도록 만든 도구 모음입니다. 파일·입력값은 외부 서버로 전송되지 않으며, 모든 계산·변환·처리는 사용자 기기에서만 이뤄집니다.

사용자가 도구를 쓸 때 외에는 별도 데이터를 수집·전송하지 않습니다.

운영자

TayLee · Software Developer

백엔드·인프라 위주의 소프트웨어 개발자. 평일은 직장에서 시스템 설계·운영을 담당하고, 주말·퇴근 후에 본 사이트를 운영합니다.

실생활 needs(연봉 실수령액·webp 변환·BMI·증명사진 등)에 맞춰 도구를 직접 만들고 유지보수합니다. 도구 build 기준 (5필터): ① 매년 유지보수 부담 없을 것 ② 공식·표준 1:1 매칭 가능 ③ 차별화 가능 ④ 2년 후도 유효 ⑤ 사이트 정체성 정합.

기술 스택

  • 이미지 처리: HTML Canvas API + browser native decoder (WebP·AVIF·HEIC)
  • PDF 처리: pdf-lib WebAssembly (브라우저 내 PDF 편집·합치기·분할)
  • 동영상 처리: ffmpeg.wasm (브라우저 내 인코딩·디코딩)
  • OCR: Tesseract.js v5 WebAssembly
  • MP3 인코딩: ffmpeg libmp3lame
  • PWA: Service Worker 캐싱, 홈 화면 설치 가능
  • 분석: Google Analytics 4 + Cloudflare Web Analytics (익명 통계)

한국판 계산기 근거 · 출처

한국 시장의 세금·금융·노무·건강 계산기는 공식 법령·기관 기준에 맞춰 작성됩니다. 요율·공제·상한은 매년 변경되며, 정확한 수치는 공식 사이트를 함께 확인 권장합니다.

  • 4대보험 (국민연금·건강보험·장기요양·고용보험) — 국민연금공단·국민건강보험공단·근로복지공단 공식 요율표
  • 소득세·연말정산·종합소득세 — 국세청 소득세법 (누진세율·인적공제·신용카드 공제율)
  • 퇴직금·연차휴가·주휴수당·실업급여·육아휴직 — 근로기준법·고용보험법
  • 최저시급 — 최저임금위원회 매년 고시
  • 자동차세·재산세·종합부동산세·취득세·증여세·상속세 — 지방세법·상속세 및 증여세법
  • 부가가치세 — 부가가치세법
  • 부동산 중개수수료·전월세 환산 — 공인중개사법·주택임대차보호법
  • BMI·비만도 — 대한비만학회 6단계 분류 + WHO 아시아 기준
  • 체지방률 — U.S. Navy method
  • 이상체중 — BMI 22 기준 + Broca·Devine·Robinson·Miller 공식
  • 기초대사량 (BMR/TDEE) — Mifflin-St Jeor 공식
  • 심박수 존 — Karvonen 공식 5 zone
  • 분유량 — 미국소아과학회 (AAP) 권장 (체중 1kg당 150~180ml/일)
  • 아기 성장 백분위 — WHO 표준 성장 곡선
  • 임신 주수·배란일 — 의학적 산정 (Naegele's rule 변형)

⚠️ 모든 계산은 참고용 추정치입니다. 세율·요율·공제율은 매년 변경되며 본 사이트는 매년 갱신을 보장하지 않습니다. 중요 의사결정(세금 신고·법적 절차·의료)은 반드시 공식 기관(국세청·국민건강보험공단·근로복지공단·해당 의료기관) 또는 전문가 확인 권장합니다. 각 도구 페이지 하단 "참고" 섹션에 도구별 상세 출처가 명시되어 있습니다.

FAQ

파일을 업로드하면 서버에 저장되나요?

아니오. 본 사이트는 백엔드 서버가 없습니다. 모든 파일 처리는 사용자 브라우저 안에서만 이뤄지며, 외부로 전송되지 않습니다. 사이트 운영자도 사용자가 처리한 파일을 볼 수 없습니다.

계산기 결과를 그대로 믿어도 되나요?

참고용 추정치입니다. 세율·법령·요율 수치는 매년 변동되며 본 사이트는 매년 갱신을 보장하지 않습니다. 중요한 결정(세금 신고·법적 절차)은 반드시 공식 기관(국세청·국민건강보험공단·근로복지공단 등) 또는 전문가 확인을 권장합니다.

광고가 보이지 않거나 너무 많아 불편해요

광고 수익으로 사이트를 운영합니다. 광고 차단 프로그램을 사용하면 광고가 보이지 않습니다. 광고 위치·빈도는 도구 사용 흐름을 방해하지 않도록 조정 중입니다.

새 도구 추가 요청은 어떻게 하나요?

위 이메일(feedback@taystudios.com)로 도구 이름·use case·기대 동작을 적어주시면 검토합니다. 단 5필터(유지보수·신뢰·차별화·2년후·정체성) 통과 도구 위주로 신중 확장하기 때문에 모든 요청이 받아들여지지는 않습니다.
