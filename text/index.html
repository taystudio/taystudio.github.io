한↔영 키보드 변환 — 어디까지 정확한가요?

두벌식 표준 키보드 매핑 + 한글 조합형 자모 상태 머신. 복합 종성 (ㄳ ㄵ ㄶ ㄺ ㄻ ㄼ ㄽ ㄾ ㄿ ㅀ ㅄ) 11종 + 복합 모음 (ㅘ ㅙ ㅚ ㅝ ㅞ ㅟ ㅢ) 7종 처리. dkssudgktpdy → 안녕하세요, gktpdy → 하세요 정확.

SNS 글 정리 — 인스타 줄바꿈 강제 방법

인스타그램은 빈 줄을 자동 압축합니다. 본 도구는 U+3164(Hangul Filler·invisible 공백)을 빈 줄에 자동 삽입해 줄바꿈을 보존합니다. 인스타에 붙여넣으면 빈 줄이 그대로 표시됩니다. 해시태그 자동 정리(중복·순서·앞뒤 #), fancy 글씨체 7종(𝓢𝓬𝓻𝓲𝓹𝓽·𝐁𝐨𝐥𝐝·𝙼𝚘𝚗𝚘 등) Unicode 변환 지원.

자주 묻는 질문

한↔영 키보드 변환은 어떻게 동작하나요?
두벌식 표준 키보드 매핑을 기반으로 합니다. "dkssudgktpdy" 같은 영문 입력을 "안녕하세요"로 자모 합성/분해 상태 머신으로 정확히 변환합니다. 복합 종성(ㄳ ㄵ 등)·복합 모음(ㅘ ㅙ 등)도 처리합니다. 단 세벌식·드보락 등 비표준 키보드는 지원하지 않습니다.
인스타 줄바꿈이 왜 안 먹히나요? 어떻게 우회하나요?
인스타그램은 단순 빈 줄을 자동으로 압축합니다. SNS 글 정리 도구는 U+3164(채움 문자)를 빈 줄에 자동 삽입해 줄바꿈이 보존되도록 처리합니다. 복사한 글을 인스타에 붙여넣으면 줄바꿈이 그대로 표시됩니다.
SNS 글자수 한도가 플랫폼마다 다른 이유는?
인스타 캡션 2200자, X(트위터) 280자, 페이스북 63K, 카톡 카드뉴스 한정 등 플랫폼별 정책 차이입니다. 글자수 카운터는 11개 SNS·자소서 한도를 progress bar로 동시 표시합니다.
입력한 텍스트가 어디로 가나요?
사용자 브라우저 메모리에서만 처리되며 외부 서버로 전송되지 않습니다. 자소서·이력서·민감 정보 텍스트도 안전합니다.
