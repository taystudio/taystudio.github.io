자주 묻는 질문

이미지가 외부 서버로 업로드되나요?
전송되지 않습니다. 모든 처리는 브라우저 Canvas API에서 이뤄집니다.
워터마크는 얼마나 강하게 도용을 막나요?
포토샵으로 크롭·블러·복원할 수는 있지만 시간이 들어 대부분의 도용을 막습니다. 강력한 보호는 (1) 반복 패턴(타일) (2) 사진 중앙 가로지름 (3) 적당한 투명도(40~60%) 조합. 본 도구는 모두 지원.
텍스트와 로고 이미지 둘 다 워터마크로 쓸 수 있나요?
네. 모드 전환으로 (1) 텍스트(폰트·크기·색상) (2) 로고 이미지(투명 PNG 권장) 중 선택.
여러 사진에 같은 워터마크를 한 번에 적용할 수 있나요?
네. 여러 파일을 드래그·드롭 후 워터마크 설정 1번만 → 일괄 적용. 미리보기로 위치 확인 가능.
원본 화질은 유지되나요?
원본 픽셀 그대로에 워터마크만 합성. JPG 출력 시 품질 95 권장. PNG 출력 시 무손실.
