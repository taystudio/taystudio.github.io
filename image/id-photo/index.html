mm를 픽셀로 직접 계산해 보기

이 도구의 출력 픽셀은 사람이 정한 게 아니라 딱 하나의 공식으로 자동 계산됩니다. 위 표의 숫자가 어디서 나왔는지 직접 검산할 수 있습니다.

단계계산
1 inch = 25.4 mm (고정)—
300 DPI = 1 inch에 300 px1 mm = 300 ÷ 25.4 ≈ 11.811 px
가로 35 mm35 × 11.811 ≈ 413.4 → 413 px
세로 45 mm45 × 11.811 ≈ 531.5 → 531 px
결과 (여권 규격)413 × 531 px

소수점은 가까운 정수로 반올림합니다. 그래서 30×40mm는 354×472px, 51×51mm(미국 비자)는 602×602px, 33×48mm(중국 비자)는 390×567px로 떨어집니다. '직접 입력'에 원하는 mm를 넣으면 같은 공식으로 픽셀이 즉시 표시되므로, 표에 없는 사이즈도 직접 확인하며 만들 수 있습니다.

참고: frame을 크게 잡든 작게 잡든 최종 출력 픽셀은 선택한 규격 값으로 고정됩니다. 크롭한 영역이 출력 크기에 맞춰 다시 그려지기(리샘플) 때문이며, 원본을 크게 잡을수록 더 선명합니다.

여권 사진 공식 규정에 맞춰 만들 수 있나요?
본 도구는 사이즈·해상도 표준만 맞춥니다. 공식 여권 규정(머리 길이 32~36mm·정면·귀 노출·무표정·흰 배경·6개월 이내 등)은 사용자가 직접 확인해야 합니다. 외교부 여권 안내(passport.go.kr) 또는 사진관 촬영 권장. 본 도구는 이력서·자기소개서·동아리·학원·앱 가입 등 비공식 용도에 가장 적합합니다.
frame이 작아서 핸들 잡기 어려워요
모서리 4개 핸들은 14×14px이지만 터치 영역이 +8px 잡혀있습니다. 모바일에서 정밀 조정이 어려우면 frame 안쪽을 드래그해 위치만 옮기는 방법도 있습니다. 큰 화면 추천.
가이드 라인은 정확한가요?
frame 안에 표시된 가이드 라인(정수리 18%·눈 42%·턱 78%)은 일반적인 한국 증명사진 권장 위치입니다. 여권 공식 규정과 약간 다를 수 있으니 공식 용도는 외교부·각 발급 기관의 가이드라인을 직접 확인하세요. 일반 이력서·이미지·앱 가입용으로는 충분합니다.
흰 배경 옵션은 어떻게 작동하나요?
체크하면 결과 PNG의 투명 영역(알파 = 0)이 흰색으로 채워집니다. 본 도구는 단순 크롭만 하므로 원본에 투명 영역이 없으면 효과가 없습니다. 인물만 남기고 배경을 투명하게 만들려면 먼저 배경 제거(누끼) 도구로 처리한 후 본 도구로 표준 사이즈 크롭하세요.

자주 묻는 질문

사진이 외부 서버로 업로드되나요?
전송되지 않습니다. 크롭은 브라우저의 canvas API에서만 처리되며, 사이트 운영자도 사진 내용을 볼 수 없습니다. 본 도구는 단순 크롭만 수행하므로 신분 서류 사진처럼 민감한 자료도 안심하고 사용하세요.
어떤 표준 사이즈를 지원하나요?
여권(35×45mm) / 일반 증명사진·이력서·반명함(30×40mm) / 운전면허·주민등록증(35×45mm — 여권과 동일) / 미국 비자(51×51mm 정사각) / 중국 비자(33×48mm) + 직접 mm 입력. 모두 한국 공식 가이드라인 기반.
왜 픽셀 크기가 자동으로 정해지나요?
일반 인쇄 표준은 300 DPI입니다. mm 사이즈에 300/25.4 ≈ 11.81을 곱한 값이 픽셀 사이즈. 예) 35mm × 11.81 ≈ 413px, 45mm × 11.81 ≈ 531px → 35×45mm = 413×531px @ 300dpi. 사진관·인쇄소에서 받아주는 기본 해상도입니다.
얼굴 위치는 어떻게 맞추나요?
크롭 frame 안에 가이드 라인(정수리·눈·턱)이 점선으로 표시됩니다. 사진을 frame 위치에서 드래그하면 frame이 이동, 모서리 핸들로 frame 크기 조정. 머리 위에 약간 여백 + 눈은 위에서 약 40% 위치 + 턱끝은 아래쪽 가까이가 한국 일반 규정에 맞습니다.
흰 배경은 어떻게 만드나요?
본 도구는 단순 크롭만 수행합니다. 원본 배경이 흰색이 아니면 결과도 흰색이 아닙니다. 흰 배경이 필요하면 먼저 '배경 제거(누끼)'로 인물만 남긴 후 본 도구로 표준 사이즈 크롭하거나, 인물·배경이 단순한 사진을 처음부터 흰 벽 앞에서 촬영하세요. '흰 배경 옵션' 체크 시 결과 PNG의 알파 영역에만 흰색이 채워집니다.
사진관에서 인화 가능한가요?
결과 파일이 표준 사이즈·300dpi이므로 일반 사진관·인쇄소에서 인화 가능합니다. 단 정부 기관 신분증 발급용은 사진관에서 직접 촬영해야 하는 경우가 많고(여권은 6개월 이내, 사진관 표시 등 요구), 본 도구는 이력서·자기소개서·동아리·학원 신청·앱 가입 등 비공식 용도에 적합합니다. 공식 신분증용은 발급 기관 가이드라인을 직접 확인하세요.
