유효하지 않은 페이지 입력
존재하지 않는 페이지 번호(예: 100쪽 PDF에 '120') 또는 잘못된 형식은 추출 시 오류 메시지로 표시됩니다. 0이나 음수는 허용되지 않으며, 공백은 자동 제거됩니다.
역순 추출 작동 방식
시작이 끝보다 큰 범위(예: 5-1)는 역순으로 인식되어 5,4,3,2,1 순서가 됩니다. 마지막 몇 페이지를 표지로 옮기는 등의 작업에 유용.
PDF 페이지 미리보기는 안 되나요?
현재 도구는 페이지 미리보기를 제공하지 않습니다(렌더링 라이브러리 추가 ~2MB 필요). 원본 PDF를 PDF 뷰어로 먼저 열어 페이지 번호를 확인한 후 범위 입력하시기 바랍니다.
pdf-lib 라이브러리
pdf-lib (Andrew Dillon, MIT). 약 700KB, 클라이언트 100% 동작. PDF 합치기·자르기 모두 같은 라이브러리.

자주 묻는 질문

페이지 범위는 어떻게 입력하나요?
1쪽부터 시작합니다. 단일 페이지는 그냥 숫자(예: 5), 연속 범위는 하이픈(예: 3-7), 여러 묶음은 콤마(예: 1-3,5,8-10)로 구분합니다. 공백은 무시됩니다. 입력 순서대로 결과 PDF에 페이지가 배치됩니다.
전체 페이지를 1쪽씩 분리할 수 있나요?
현재는 1개의 결과 PDF로 추출합니다. 1쪽씩 N개 파일로 자동 분리하는 기능은 향후 검토 중. 지금은 페이지마다 따로 추출(예: '1' 입력 → 다운로드 → '2' 입력 → 다운로드)하시거나, 운영체제 인쇄 → 'PDF로 저장' 옵션에서 페이지 범위를 매번 다르게 지정해 분할할 수 있습니다.
원본 PDF가 외부 서버로 업로드되나요?
전송되지 않습니다. PDF 처리는 브라우저 내부의 pdf-lib 라이브러리에서 수행되며, 사이트 운영자도 파일 내용을 볼 수 없습니다. 계약서·이력서·민감 문서 안전합니다.
암호 걸린 PDF도 자르기 되나요?
암호가 걸린 PDF는 처리되지 않을 수 있습니다. PDF 뷰어에서 암호 해제 후 다시 업로드하세요. 본 도구는 암호 해제 기능을 제공하지 않습니다.
페이지 순서를 바꿀 수 있나요?
네. 입력 순서대로 결과 PDF에 페이지가 배치됩니다. 예: 5,1-3을 입력하면 결과는 '5쪽 → 1쪽 → 2쪽 → 3쪽' 순서로 4쪽짜리 PDF가 됩니다. 같은 페이지를 두 번 넣을 수도 있습니다.
결과 파일이 원본보다 용량이 크게 줄지 않는데?
pdf-lib는 페이지 객체를 그대로 복사하므로 일부 페이지만 추출해도 폰트·이미지 등 내부 자원이 포함되어 용량이 비례적으로 줄지 않을 수 있습니다. 추출 후 별도 PDF 압축 도구를 추가로 사용하면 더 줄일 수 있습니다.
