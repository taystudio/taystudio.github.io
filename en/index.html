Is it really free? Any hidden costs?

Fully free. No signup, no payment, no email required. The site is supported only by displayed ads and affiliate recommendations (Coupang Partners and similar). No tool has usage limits.

Where do my uploaded files and inputs go?

Nowhere external. Images, PDFs, videos, and calculator inputs are processed only in your browser memory and discarded after download. The server stores no inputs and keeps no logs.

We collect anonymous pageviews only (Google Analytics). File contents and calculator inputs are never sent to GA either.

Why does the calculator differ from my actual paystub?

Calculators are estimates based on official rates. Real paystubs vary with company-specific non-taxable items (meal allowance 200K KRW), company-side deductions (union dues, welfare fund), year-end adjustments, and April health-insurance reconciliation — typically within ±100K to 300K KRW. Each tool's "Sources" footer links to the law / rate gazette references.

Where can I convert iPhone HEIC files?

Use image → HEIC → JPG. Multi-page HEIC is auto-split, and 20+ batch processing shows a safety confirm. The output JPG also has EXIF GPS stripped.

Does it work on iPad and iPhone?

iOS Safari 15+ / iPadOS 15+ are fully supported. Camera-permission tools (QR scan, OCR) request permission, and if denied, you can upload images from the gallery instead. iCloud / Google Photos cloud picks may prompt re-selection if permissions expire.

What is the file size limit?

Defaults: 100MB for images and PDFs, 1GB for videos. Actual limits depend on device memory. Video tools pre-estimate memory and block via confirm if risky.

How is the site updated when laws or rates change?

Each tool's "Sources" footer lists the last verification date. We sweep all calculators every November (gazette season) and January (new fiscal year), and we update immediately on major changes (national pension hike, minimum wage). Change history is in About.

Found a bug or mis-calculation?

Email thlee991@gmail.com with the source link, and we'll verify against the official page and update. Policy: tax / rate numbers are always sourced from the issuing agency directly — no blog or news quotes.
