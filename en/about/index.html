About

TAYSTUDIO is a solo-operated site providing 60+ free utility tools. All processing in your browser.

Last updated: 2026-05-13

Mission

A collection of practical utility tools for everyday tasks — calculators, image, PDF, video, and text processing. Everything runs entirely in your browser. Files and inputs are never sent to any server; all computation happens on your device.

No data collection beyond standard anonymized web analytics.

Operator

TayLee · Software Developer

Software developer with a backend and infrastructure focus. Day job: systems design and operations. Side project: this site, built and maintained on evenings and weekends.

Tools cover real-world utility tasks — calculators, image, PDF, video, and text processing. English mirrors for globally relevant tools. Tool build criteria (5 filters): ① No yearly maintenance burden ② 1:1 match with official sources ③ Real differentiation ④ Still useful in 2 years ⑤ Identity alignment.

Tech Stack

  • Image processing: HTML Canvas API + browser native decoders (WebP, AVIF, HEIC)
  • PDF processing: pdf-lib WebAssembly (in-browser PDF edit/merge/split)
  • Video processing: ffmpeg.wasm (in-browser encoding/decoding)
  • OCR: Tesseract.js v5 WebAssembly
  • MP3 encoding: ffmpeg libmp3lame
  • PWA: Service Worker caching, installable as home screen app
  • Analytics: Google Analytics 4 + Cloudflare Web Analytics (anonymous stats)

Calculator References

Calculators on the English mirror are based on peer-reviewed formulas and international standards.

  • BMI — WHO Asia-Pacific 6-stage classification (underweight, normal, overweight, obese class 1/2/3)
  • Body Fat — U.S. Navy method (circumference measurements)
  • Ideal Weight — Broca, Devine, Robinson, Miller formulas + BMI 22 reference
  • BMR / TDEE (Calorie) — Mifflin-St Jeor equation (1990)
  • Compound Interest — Standard compound interest formula A = P(1+r/n)^(nt)
  • Loan — Equal monthly payment (annuity) and equal principal methods
  • Savings — Periodic deposit future value formula
  • D-day — Gregorian calendar date arithmetic (day/week/month/year units)

⚠️ All calculations are reference estimates only. Formulas may have edge cases (athletes for BMI, extreme age ranges, etc.). Verify with professionals (doctors, financial advisors) for important decisions. Each tool page has a "References" section at the bottom with detailed sources.

FAQ

Are files uploaded to a server?

No. This site has no backend server. All file processing happens entirely in your browser. The operator cannot see what you process — there is no upload.

Can I trust the calculator results?

Calculator results are reference estimates. Tax rates and rules change yearly, and this site doesn't guarantee yearly updates. For important decisions (tax filing, legal matters), please verify with official sources (NTS, NHIS, KEIC) or professional advisors.

Why do I see ads / no ads?

Ad revenue funds the site. Ad blockers will hide them. Ad placement is tuned to avoid disrupting tool flow.

How to request a new tool?

Open a GitHub Issue with the tool name, use case, and expected behavior. Note: only tools passing the 5-filter criteria are accepted, so not every request will be built.
