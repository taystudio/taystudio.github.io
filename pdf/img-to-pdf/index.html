왜 PNG보다 JPG가 PDF 용량이 작은가요?
JPG는 손실 압축 포맷이라 사진 데이터를 효율적으로 압축합니다. PNG는 무손실이라 같은 사진이어도 용량이 5~10배 큽니다. 본 도구는 JPG는 그대로 임베드, PNG는 그대로 임베드(PDF 내부에서 PNG 압축 적용). 텍스트·로고가 아닌 사진이라면 미리 이미지 압축 도구로 JPG 변환을 권장.
EXIF 회전이 적용되나요?
현재는 브라우저가 디코딩한 픽셀 그대로 PDF에 들어갑니다. 일부 폰 사진은 EXIF rotation 메타가 있어도 픽셀은 회전 안 된 채로 저장되는 경우가 있는데, 그럴 때 PDF에서 이미지가 옆으로 누워 보일 수 있습니다. 그 경우 운영체제 사진 앱에서 회전·저장 후 다시 업로드하세요.
대용량 처리 팁
① 미리 이미지 리사이즈로 가로 1500~2000px로 줄이기 ② JPG로 변환·압축 ③ PDF 변환 → 결과 용량 크게 감소. 모바일에서 30장+ 처리는 메모리 부족 가능, 데스크톱 권장.
pdf-lib 라이브러리
pdf-lib (Andrew Dillon, MIT). PDF 합치기·자르기·이미지 → PDF 모두 같은 라이브러리. 본 사이트는 vendoring(직접 commit)으로 외부 CDN 의존 없음.

자주 묻는 질문

이미지가 외부 서버로 업로드되나요?
전송되지 않습니다. PDF 변환은 브라우저 내부의 pdf-lib 라이브러리에서 모두 수행되며, 사이트 운영자도 이미지 내용을 볼 수 없습니다. 신분증·이력서·계약서 사진처럼 민감 자료도 안심하고 사용하세요.
어떤 이미지 형식을 지원하나요?
JPG·PNG가 PDF에 직접 임베드됩니다. WebP·GIF·BMP는 자동으로 PNG로 변환된 후 PDF에 들어갑니다. 모든 일반 이미지 포맷이 가능하며, JPG가 PDF 용량 측면에서 가장 효율적입니다.
이미지 순서를 바꿀 수 있나요?
네. 업로드된 이미지 목록에서 ↑·↓ 버튼으로 순서를 조정할 수 있습니다. 위에서 아래 순서대로 PDF 페이지가 생성됩니다. 필요 없는 이미지는 ✕ 버튼으로 제거.
페이지 크기는 어떻게 정해지나요?
A4·Letter·원본 크기 중 선택. A4·Letter는 표준 인쇄 페이지에 맞춰 이미지가 비율 유지하며 중앙 배치됩니다. '원본 크기'는 각 이미지 픽셀 크기 그대로 PDF 페이지가 만들어져 디지털 보관에 적합합니다.
최대 몇 장까지 한 PDF에 담을 수 있나요?
이론적으로 제한 없으나 브라우저 메모리에 좌우됩니다. 데스크톱 Chrome 기준 50~100장까지 무난(한 장당 평균 1~2MB 가정). 모바일은 20~30장 권장. 큰 사진은 미리 이미지 압축 도구로 줄여서 PDF 변환하면 빠르고 안전합니다.
결과 PDF가 너무 커요. 줄일 수 있나요?
JPG가 PNG보다 PDF 용량이 작습니다. 원본이 PNG라면 본 사이트의 이미지 압축 도구로 JPG로 변환한 후 PDF 만들기를 권장. 또한 이미지 리사이즈로 해상도를 1500~2000px로 줄이면 PDF 용량이 크게 줄어듭니다.
