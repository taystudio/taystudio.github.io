PDF 편집·주석

PDF 위에 텍스트·펜·형광펜·도형을 얹고 흰색으로 가립니다. 한글·영문 글꼴을 지원하고, 모든 처리는 브라우저 안에서만 일어납니다.

🔒 입력 PDF는 외부로 전송되지 않습니다. pdf-lib·pdf.js JavaScript 라이브러리로 브라우저 안에서만 처리됩니다.
⚠️ 이미 박혀 있는 글자를 직접 고치는 게 아니라 위에 덧입히는 방식입니다. 기존 내용을 바꾸려면 가림(흰색)으로 덮고 그 위에 텍스트를 얹으세요.

PDF 편집·주석이란?

출력해서 펜으로 적고 다시 스캔하던 일을 PDF 위에서 바로 하는 도구입니다. 빈 양식에 글자를 채우고, 형광펜으로 강조하고, 도장·서명 자리를 표시하고, 바꿔야 할 부분은 흰색으로 가린 뒤 새 글자를 얹습니다. 신청서·계약서·과제·견적서처럼 "몇 글자만 더하거나 가리면 되는" 문서에 특히 편리합니다.

모든 처리는 사용자 브라우저 안에서 이뤄지며 파일이 외부 서버로 전송되지 않아, 신분 관련 서류나 계약서도 안심하고 다룰 수 있습니다.

사용 방법 — 4단계

  1. PDF 선택 — 편집할 PDF를 불러오면 미리보기가 표시됩니다. 여러 페이지면 ← → 로 이동.
  2. 도구 선택 — 텍스트·펜·형광펜·사각형·가림(흰색) 중에서 고르고, 색·글꼴·크기·두께를 정합니다.
  3. 편집 — PDF 위를 클릭·드래그해 글자를 넣거나 그립니다. 텍스트 박스를 클릭하면 크기·색·글꼴을 즉시 바꾸고 ✛로 옮길 수 있습니다.
  4. 다운로드 — 편집이 적용된 새 PDF를 내려받습니다. 원본은 그대로입니다.

이런 상황에 씁니다

  • 신청서·양식 채우기인쇄 없이 빈칸에 이름·날짜·내용을 바로 입력
  • 오타·내용 수정틀린 부분을 흰색으로 가리고 그 위에 새 글자
  • 민감정보 가림주민번호·계좌번호 등을 흰색으로 덮어 제출
  • 강조·검토형광펜·사각형·펜으로 중요한 곳 표시
  • 도장·서명 자리텍스트·손글씨 글꼴로 표시 (서명은 전자서명 도구도)
  • 과제·결재코멘트를 얹어 회신

지원 글꼴

한글 — 나눔고딕(고딕)·고운돋움(돋움)·나눔명조(명조/바탕)·나눔손글씨(손글씨)·검은고딕(굵은 제목)·도현(둥근 고딕). 영문 — Helvetica·Lato·Times New Roman·Crimson·Courier·Bebas Neue·Great Vibes(서명체)·Pacifico. 선택한 글꼴은 출력 PDF에 함께 포함되어 다른 기기에서도 동일하게 보입니다.

개인정보·보안

  • 외부 업로드 없음 — PDF는 브라우저 메모리에서만 처리됩니다. pdf-lib·pdf.js가 클라이언트 사이드로 동작.
  • 서버 저장 없음 — 운영자도 파일 내용을 볼 수 없습니다. 탭을 닫으면 메모리에서 사라집니다.
  • 가림 주의 — 흰색 가림은 위에 덮는 방식입니다. 추출 차단이 중요하면 가린 영역이 글자를 완전히 덮는지 확인하세요.

자주 묻는 질문

PDF가 외부 서버로 업로드되나요?
전송되지 않습니다. 브라우저의 pdf-lib·pdf.js JavaScript 라이브러리에서만 처리하며, 운영자도 PDF 내용을 볼 수 없습니다.
이미 있는 글자를 직접 고칠 수 있나요?
PDF는 글자가 좌표에 고정돼 있어 기존 글자를 직접 수정·재배치하는 것은 불가능합니다. 대신 흰색 가림으로 덮은 뒤 그 위에 새 텍스트를 얹는 방식으로 수정합니다.
한글 글꼴도 지원하나요?
네. 나눔고딕·고운돋움·나눔명조·나눔손글씨·검은고딕·도현 등 한글 글꼴과 Helvetica·Times New Roman·Courier 등 영문 글꼴을 지원하며, 선택한 글꼴이 출력 PDF에 함께 포함됩니다.
민감정보를 가릴 수 있나요?
흰색 가림 도구로 주민번호·계좌번호 등을 덮을 수 있습니다. 시각적으로 가리는 방식이므로, 원본 텍스트가 추출되지 않도록 하려면 가린 영역이 글자를 완전히 덮는지 확인하세요.
그린 선이나 주석을 지울 수 있나요?
네. 지우개로 펜·형광펜을 드래그하면 지나간 부분만 지워지고(부분 지우기), 영역 선택(네모)·올가미(자유 곡선)로 여러 주석을 묶은 뒤 Delete 키로 한 번에 삭제할 수 있습니다. 텍스트 박스는 × 버튼으로도 지웁니다.
