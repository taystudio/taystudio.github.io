PDF 전자서명이란?

종이 문서에 도장을 찍거나 펜으로 사인하던 일을 PDF 파일 위에서 그대로 하는 것입니다. 출력 → 서명 → 다시 스캔하는 번거로운 과정 없이, PDF에 서명 이미지를 직접 얹어 완성합니다. 서약서·동의서·간단한 계약서·민원 신청서처럼 "서명만 더하면 끝" 인 문서에 특히 편리합니다.

이 도구는 서명을 이미지로 PDF 안에 삽입하는 방식입니다. 원본 PDF의 텍스트·표·이미지는 그대로 두고, 사용자가 만든 서명만 원하는 위치에 얹습니다. 모든 처리는 사용자 브라우저 안에서 이뤄지며, 파일이 외부 서버로 전송되지 않습니다 — 계약서·신분 관련 문서도 안심하고 사용할 수 있습니다.

사용 방법 — 4단계

  1. PDF 선택 — 서명할 PDF 파일을 불러오면 미리보기가 표시됩니다. 여러 페이지면 ← → 로 이동.
  2. 서명 만들기 — 마우스·손가락으로 직접 그리거나, 이름을 텍스트로 입력하거나, 미리 만든 서명 이미지를 업로드합니다.
  3. 위치·크기 조정 — PDF 위에 나타난 서명을 드래그해 원하는 칸으로 옮기고, 우하단 모서리 핸들을 끌어 크기를 맞춥니다. 투명도도 조절 가능.
  4. 다운로드 — 서명이 박힌 PDF를 내려받습니다. 원본은 그대로, 서명만 추가된 새 파일입니다.

이런 상황에 씁니다

  • 서약서·동의서 — 인쇄·스캔 없이 PDF에 바로 서명해서 회신
  • 내부 결재·간단한 계약 — 자필 서명을 그려 삽입, 메일로 즉시 전달
  • 민원·신청서 — 이름 텍스트 서명을 빠르게 얹어 제출
  • 임대·거래 확인서 — 도장 이미지(PNG)를 한 번 만들어 두고 반복 사용
  • 학교·학원 제출 서류 — 보호자 서명을 폰·태블릿에서 손가락으로
  • 프리랜서 견적·인보이스 — 발행 문서에 서명 박아 신뢰도 ↑

전자서명 vs 공인 전자서명 — 무엇이 다른가

이 도구가 만드는 것은 "시각적 전자서명" 입니다. 서명 이미지를 PDF에 얹어 "누가 동의했다" 는 표시를 남기는 것으로, 일상적인 서약서·내부 결재·간단한 동의서에는 충분합니다.

반면 공인 전자서명(디지털 서명) 은 PKI 인증서·타임스탬프로 "위·변조되지 않았음"과 "서명 시점"을 암호학적으로 증명합니다. 부동산 계약·법적 분쟁 가능성이 있는 중요 계약은 공인 전자서명 인증 서비스를 사용하는 것이 안전합니다.

위·변조 증명불가가능
비용무료유료 (인증 서비스)
적합한 문서서약서·동의서·내부 결재부동산·중요 계약

개인정보·보안

  • 외부 업로드 없음 — PDF와 서명은 사용자 브라우저 메모리에서만 처리됩니다. pdf-lib·pdf.js JavaScript 라이브러리가 클라이언트 사이드로 동작.
  • 서버 저장 없음 — 운영자도 파일 내용을 볼 수 없습니다. 작업 후 탭을 닫으면 메모리에서 사라집니다.
  • 네트워크 확인 — 브라우저 개발자도구의 Network 탭에서 파일 업로드 요청이 없음을 직접 확인할 수 있습니다.

자주 묻는 질문

PDF가 외부 서버로 업로드되나요?
전송되지 않습니다. 브라우저의 pdf-lib·pdf.js JavaScript 라이브러리에서만 처리하며, 운영자도 PDF 내용을 볼 수 없습니다.
전자서명이 법적 효력이 있나요?
본 도구는 서명 이미지를 PDF에 시각적으로 삽입하는 도구입니다. 공인전자서명(PKI)·타임스탬프 기능은 없습니다. 법적 효력이 중요한 계약은 전자서명 인증 서비스를 권장합니다. 일반 서약서·내부 결재·간단한 동의서에는 충분합니다.
서명을 여러 방식으로 만들 수 있나요?
네. (1) 자필 그리기 (2) 텍스트 입력 (필기체·세리프·고딕) (3) 서명 이미지 업로드 — 3가지를 지원합니다.
위치와 크기를 조정할 수 있나요?
네. PDF 미리보기 위에서 서명을 드래그해 위치를 맞추고, 우하단 모서리 핸들로 크기를 조정합니다. 투명도도 조정 가능합니다.
여러 페이지 중 원하는 페이지에 서명할 수 있나요?
네. 페이지 이동(← →) 후 원하는 페이지에서 서명 위치를 잡고 다운로드하면 해당 페이지에 서명이 삽입됩니다.
