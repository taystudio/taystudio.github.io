자주 묻는 질문

세율은 언제 기준인가요?
각 도구는 출시·최종 검증 시점의 공식 세율 기준입니다. 세율 변경 시 도구 페이지의 .updated 줄과 .sources 출처 링크가 함께 갱신됩니다. 계산 결과 옆 갱신일을 확인 후 사용하세요.
이 계산 결과로 세무 신고를 해도 되나요?
아니요. 모든 계산은 참고용 추정치입니다. 실제 양도세·종부세·연말정산 등 신고는 국세청 홈택스 또는 세무 전문가를 통해 정확히 산출하세요. 사이트는 통상적인 케이스 위주로 단순화돼 있어 특수 공제·중과 케이스가 빠질 수 있습니다.
임신·육아·건강 계산기는 의료 진단을 대체하나요?
아니요. BMI·기초대사량·체지방률·심박존·출산예정일 등은 일반 가이드라인 기반 추정입니다. 의사·산부인과·전문 트레이너 진단·상담이 필요한 경우 공식 의료기관을 이용하세요.

사용자 브라우저 메모리 안에서만 처리되며 외부 서버로 전송되지 않습니다. 페이지를 새로고침하거나 닫으면 입력값은 자동 폐기됩니다. 일부 도구는 사용자 편의를 위해 LocalStorage에 임시 저장할 수 있습니다.
