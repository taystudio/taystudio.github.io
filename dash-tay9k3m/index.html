라이브 데이터 — Looker Studio

embed · 본인 Google 로그인 시에만 노출
설정 방법 (~5분) · 보안 주의사항
  1. lookerstudio.google.com 접속 → 빈 보고서 만들기 → 데이터 소스 = Google Analytics → taystudio.github.io 속성 선택
  2. 차트 추가 (예: 실시간 활성 사용자 / 페이지뷰 / 페이지별 클릭 / 유입 채널)
  3. 우측 상단 공유 → 일반 액세스를 제한됨(Restricted)으로 두고 본인 Gmail만 추가. 절대 "링크가 있는 모든 사용자"로 두지 말 것
  4. 우측 상단 파일 → 보고서 임베드 → 임베드 사용 설정 토글 ON → iframe 임베드 코드 복사 → src="..." URL만 추출
  5. 이 파일 (dash-tay9k3m/index.html)에서 <!-- iframe --> 주석 블록 해제 + src에 URL 붙여넣기 → 위 "보고서 미설정" 박스 삭제
보안 체크
① 공유 = "특정 사용자만 + 본인 Gmail" 외 옵션은 GA 데이터 노출 위험.
② referrerpolicy="no-referrer" 유지 — secret URL(/dash-tay9k3m/)이 Google referer 로그에 안 잡힘.
③ 본인이 다른 브라우저·시크릿 모드·로그아웃 상태로 dash 열면 iframe = 빈 화면/로그인 화면. 정상 동작.
④ Looker가 iframe 내 로그인을 차단하는 경우 → 새 탭에서 lookerstudio.google.com 직접 열어 한 번 로그인 후 dash 새로고침.
